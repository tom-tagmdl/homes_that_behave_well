# Scenario: Room Change Summary

## Purpose

Summarize meaningful changes over time.

---

## Scenario

User asks:

"What changed in this room today?"

---

## System Behavior

System filters events:

- risk transitions
- environment changes
- configuration changes

---

## Output

"Today, humidity increased above the recommended range in the afternoon, causing the piano to transition to AMBER. Conditions returned to normal in the evening."

---

## Key Principles

- summarize only meaningful changes
- avoid noise
- maintain clarity and brevity

---

## Final Outcome

User quickly understands what changed.
